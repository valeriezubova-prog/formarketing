# Telegram бот "да / нет / не знаю"

Простой Telegram-бот для рабочих и дружеских чатов.  
Реагирует только на строго заданные фразы и не флудит.

## Логика бота

Если пользователь в чате пишет:
- `да` → бот отвечает: `в кране черная вода!`
- `нет` → бот отвечает: `с-маркетинга ответ!`
- `не знаю` → бот отвечает: `а кто знает?!`

Регистр и пробелы вокруг фразы игнорируются.  
Любые другие сообщения бот игнорирует.

## Переменные окружения

Нужно задать:
- `BOT_TOKEN` — токен Telegram-бота от BotFather
- `WEBHOOK_BASE_URL` — публичный URL приложения (например, адрес сервиса на Render)
